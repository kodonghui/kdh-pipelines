# Corthex Console — UI Kit

A cosmetic, interactive recreation of the corthex console for design prototyping. Mirrors the component structure in `packages/app/src/` of `kodonghui/corthex-v3`.

- **Source of truth:** DESIGN.md + `packages/app/src/pages/hub.tsx`, `packages/app/src/components/chat/*`, `packages/app/src/components/layout/*`
- **Theme:** Paper default; theme switcher (Paper / Carbon / Signal) bottom-left
- **Screens demoed:** Hub (agent list + chat), Profile, Session-expired dialog

Open `index.html` directly. Components live in sibling `.jsx` files and are loaded as Babel-transpiled scripts.
